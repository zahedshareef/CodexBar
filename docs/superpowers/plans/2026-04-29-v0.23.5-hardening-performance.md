# v0.23.5 Hardening and Performance Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship Win-CodexBar v0.23.5 with faster provider refresh behavior, stronger secret/logging protections, safer update/install flows, and clearer diagnostics without removing existing app functionality.

**Architecture:** Keep the existing Tauri shell and shared Rust backend shape. Add small backend modules for refresh policy, redaction, diagnostics, and installer/update verification, then wire them through the current command/event bridge with focused frontend state changes.

**Tech Stack:** Rust 2024, Tauri 2, React 18, TypeScript, Vitest, Cargo tests, Windows DPAPI, Inno Setup installer metadata, GitHub release assets.

---

## Release Scope

This is intentionally one release train, but implementation must stay split into independent commits:

1. Refresh performance and stale-cache UX.
2. Provider timing instrumentation.
3. Secret redaction and safe diagnostics.
4. Credential lifecycle and migration status.
5. Tauri command-surface input hardening.
6. Update download SHA-256 verification.
7. Installer smoke-test script and release docs.
8. Version bump and Windows release validation.

Do not add new providers in this release. Do not change provider semantics unless required for security or performance.

## Task 9: Installer Smoke Test Script

**Files:**

- Modify: `docs/WINDOWS_PROOF.md`

- [ ] **Step 1: Create smoke test script**

Script behavior:

```powershell
param(
  [Parameter(Mandatory=$true)][string]$InstallerPath,
  [string]$ExpectedVersion = "0.23.5"
)

$ErrorActionPreference = "Stop"
$installArgs = @("/VERYSILENT", "/SUPPRESSMSGBOXES", "/NORESTART")
$p = Start-Process -FilePath $InstallerPath -ArgumentList $installArgs -Wait -PassThru
if ($p.ExitCode -ne 0) { throw "Installer failed with exit code $($p.ExitCode)" }

$entry = Get-ItemProperty "HKCU:\Software\Microsoft\Windows\CurrentVersion\Uninstall\*" |
  Where-Object { $_.PSChildName -eq "WinCodexBar_is1" } |
  Select-Object -First 1

if (-not $entry) { throw "WinCodexBar_is1 uninstall entry not found" }
if ($entry.DisplayVersion -ne $ExpectedVersion) {
  throw "Expected version $ExpectedVersion, got $($entry.DisplayVersion)"
}
if (-not (Test-Path (Join-Path $entry.InstallLocation "codexbar.exe"))) {
  throw "codexbar.exe not found in install location"
}

$uninstall = $entry.QuietUninstallString
Start-Process -FilePath "cmd.exe" -ArgumentList "/c $uninstall" -Wait
```
